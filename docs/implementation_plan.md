# 무적의 오토파일럿 (Perfect Autopilot) AI 강화 계획

현재의 단순 거리 기반 오토파일럿을 고도화하여, 게임 속도 증가와 장애물의 연속 배치에 관계없이 **절대 피해를 입지 않는 무적의 AI**로 업그레이드합니다.

## User Review Required

> [!IMPORTANT]
> 1. **물리 기반 계산 (TTC Logic)**: 단순히 선인장의 거리가 아닌, 현재 속도를 고려한 **충돌 예상 시간(Time To Collision, TTC)**을 매 프레임 계산하여 최적의 점프 타이밍을 결정합니다.
> 2. **지능형 체이닝 (Jump Chaining)**: 선인장이 연속으로 올 경우, 점프 키를 계속 누르고 있어 체공 시간을 극대화하거나(Variable Jump 활용), 착지 직후 즉시 재점프하는 로직을 추가합니다.
> 3. **안전 제일 원칙**: 고양이의 히트박스와 선인장의 판정을 미리 시뮬레이션하여 0.01%의 확률도 놓치지 않고 장애물을 회피합니다.
