# 다이나믹 요소 업데이트 계획 (큰 선인장, 쥐, 동적 변수)

기존 게임에 패턴의 다양성과 아이템 회복 메커니즘을 추가하여 더욱 다이나믹하고 재미있는 런닝 게임으로 진화시킵니다.

## User Review Required

> [!IMPORTANT]
> - **다이나믹 변수 (가변 점프 / 랜덤성)**: 스페이스바를 누르는 시간에 따라 점프 높이가 달라지는 "가변 점프" 시스템을 도입하여 조작감을 높일 계획입니다. 또한 장애물(선인장) 스폰 주기와 종류를 랜덤하게 배치하여 긴장감을 유도하겠습니다.
> - **쥐(Mouse) 아이템**: 피하기만 해야하는 선인장 사이사이에 먹으면 체력이 오르는 아주 귀여운 쥐 픽셀아트를 만듭니다. (체력은 최대 5개의 하트를 넘지 않도록 제한합니다.)

### 3. 게임 로직 고도화 (다이나믹 시스템, 가변 점프)

#### [MODIFY] [src/hooks/useGameLoop.ts](file:///home/rheehoselenovo2/개발프로젝트/Goyangirun/src/hooks/useGameLoop.ts)
- **가변 점프(Variable Jump)**: `isJumping` 상태일 때 사용자가 키보드(또는 터치)를 떼기 전까지는 중력을 조금 약하게 받아 살짝 길게 점프하고, 키를 떼면 중력이 강해져 빨리 떨어지도록 하는 로직을 도입합니다.
- **엔티티 구분**: 기존 `obstacles` 배열을 확장하여 오브젝트의 `type` 필드를 만듭니다 (`CACTUS_SMALL`, `CACTUS_LARGE`, `MOUSE`). 
- **랜덤 스폰 로직**: 스폰 시 정해진 확률로 작은 선인장, 큰 선인장, 혹은 쥐를 스폰합니다. 스폰 간격도 게임 속도에 따라 조금씩 랜덤성(Variance)을 부여해 갑자기 연달아 나오거나 멀리 떨어져 나오는 등 다이나믹함을 줍니다.
- **쥐 회복 로직**: 쥐 오브젝트와 겹쳤을 때 충돌은 하되, 무적 프레임 및 데미지 로직을 타는 대신 쥐를 삭제하고, `hp`를 `+3` (하트 1개 분량) 올려주며 `playHealSound()`를 재생합니다. (최대 체력 15 초과 불가).
