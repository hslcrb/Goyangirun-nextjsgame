# 고양이런 (Cat Run) 디테일 업데이트 완료! 🎀

요청하신 귀여운 핑크 테마와 정밀한 체력 시스템, 그리고 직접 작곡한 주파수 기반 효과음이 게임에 성공적으로 반영되었습니다.

## 새롭게 추가된 기능들

1. **상큼한 핑크UI 및 점프 귀요미 모션**
   - 배경, 버튼, 글자 톤을 모두 화사한 핑크색(`bg-pink-100` 등)으로 변경했습니다.
   - 고양이가 점프할 때 윙크하고 두 손을 번쩍 드는 귀여운 모션(`CAT_JUMP` 스프라이트)을 픽셀 아트로 적용했습니다.

2. **디테일한 체력(하트) 시스템**
   - 시작 시 5개의 커다란 하트가 주어집니다.
   - 캐릭터와 선인장이 얼마나 깊게 닿았는지(`overlapArea`)를 계산하여, 살짝 스치면 **하트가 1/3만 (가운데가 약간 파인 모양)** 깎이고 정면 충돌하면 크게 깎입니다.
   - 데미지를 입으면 고양이가 약 1초간 깜빡이며 추가 데미지를 받지 않는 **무적 시스템(i-frames)**이 발동합니다.
   - 피격 시, 잃은 체력만큼 작고 부서진 핑크 조각들(파티클 시스템 적용)이 떨어졌다 사라지는 뿅뿅 모션 애니메이션이 화면에 그려집니다.

3. **Web Audio API 직접 코딩 작곡 시스템 🎵**
   - `<audio>` 태그나 MP3 파일 업로드 없이, 순수 코드로 진동수(Oscillator)를 만들어 소리를 냅니다.
   - **배경음악(BGM)**: 경쾌한 150BPM 템포로 '도-미-솔-라-높은도-라-솔-미' 의 8비트 레트로 멜로디가 지속적으로 재생됩니다.
   - **점프 효과음**: 점프 시 주파수가 쭉 올라가는 귀여운 소리!
   - **피격 효과음**: 부딪혔을 때 톱니파 노이즈 소리가 우스꽝스럽게 납니다.
   - (우측 상단에 '소리 끄기/켜기' 버튼을 마련했습니다.)

> [!TIP]
> - 브라우저 보안 정책 상, 맨 처음 사용자가 키보드로 스페이스바를 누르거나 마우스로 화면 클릭을 해야만 오디오 시스템이 시작합니다! "볼륨 켜고 귀엽게 시작하기" 버튼을 눌러보세요.
> - 백그라운드에서 돌아가고 있는 기존 `npm run dev` 서버 포트(`http://localhost:3000`)에 그대로 접속하시거나 새로고침 하시면 바로 확인할 수 있습니다.
