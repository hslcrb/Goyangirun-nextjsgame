# 고양이런 (Cat Run) 구현 완료! 🐈

요청하신 "크롬 공룡 게임" 스타일의 횡스크롤 고양이 점프 게임 구현이 완료되었습니다!
HTML5의 `<canvas>` 렌더링 방식과 자체 제작한 픽셀 아트를 사용하여 향상된 성능과 귀여운 디자인을 적용했습니다.

## 주요 구현 사항

1. **커스텀 픽셀 아트 (`src/utils/assets.ts`)**
   - 배열 스니펫으로 직접 고양이(달리기 프레임 2벌)와 선인장 장애물 픽셀 데이터를 구성했습니다.
   - `drawPixelArt` 함수를 통해 캔버스 맵에 커스텀 컬러로 고양이가 예쁘게 렌더링되게 만들었습니다.

2. **게임 루프 최적화 (`src/hooks/useGameLoop.ts`)**
   - React 렌더링 사이클에 의존하지 않고 브라우저의 `requestAnimationFrame` 훅을 사용해 60FPS의 안정적인 물리 엔진(중력 적용)과 충돌 판정 로직을 만들었습니다.
   - 시간이 지남에 따라 게임이 점점 빨라지는 난이도 시스템을 추가했습니다.

3. **조작 및 UI (`src/components/Game.tsx`)**
   - **스페이스바** 또는 **화면 클릭**만으로 고양이가 점프할 수 있게 했습니다.
   - 깔끔한 반투명 UI와 애니메이션 버튼으로 게임 시작 및 오버 화면을 구현했습니다.

## 어떻게 실행하나요?

터미널에서 아래 명령어를 실행하여 게임을 켜고 브라우저에서 확인할 수 있습니다.

```bash
npm run dev
```

(제가 백그라운드 서버를 통해 실행시켜 드릴 수도 있습니다. 필요하시면 말씀해주세요!)

> [!TIP]
> - 스페이스바와 마우스 클릭 모두 점프가 가능합니다.
> - 점수가 높아질수록 선인장들이 점점 더 빠르게 다가옵니다! 플레이타임을 즐겨주세요.
> - 배경 효과나 아이템, 또는 다른 장애물이 필요하다면 언제든지 추가를 요청해 주세요.
