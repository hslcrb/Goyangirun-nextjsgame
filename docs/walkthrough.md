# 다이나믹 게임 요소 업데이트 완료! 🌵🐭

게임을 한층 업그레이드할 동적인 요소와 새로운 방해물, 아이템들이 무사히 적용되었습니다.

## 주요 업데이트 내용

1. **새로운 장애물: 왕 선인장 (Large Cactus)**
   - 기존의 선인장보다 훨씬 뚱뚱하고 뾰족한 왕 선인장이 출현합니다.
   - 피하기 어렵고, 부딪히면 데미지를 크게 입으니(1~2 하트) 각별히 주의해야 합니다!

2. **아이템: 생쥐 (Mouse)**
   - 장애물 사이로 바닥을 쏘다니는 귀여운 핑크 꼬리 생쥐가 도망다닙니다.
   - 생쥐를 잡아먹으면(부딪히면) 데미지를 입지 않고, **하트 1개를 회복**합니다! (최대 5하트는 넘지 않습니다).
   - "띠리링~" 하는 밝은 주파수 치유음 효과(`playHealSound`)가 출력됩니다.

3. **가변 점프와 다이나믹 물리 엔진**
   - **가변 점프 (Variable Jump)**: 이제 스페이스바나 화면을 톡 하고 치면 살짝 뛰지만, 꾹~ 누르고 있으면 좀 더 중력을 적게 받아 붕 뜨면서 더 멀리 뛸 수 있습니다. 상황에 맞게 큰 선인장을 넘을 때 사용해 보세요.
   - **스폰 랜덤성**: 장애물/아이템의 등장 구역과 타이밍이 게임 속도와 더불어 랜덤한 편차(Variance)를 가지도록 변경되어, 어떤 타이밍에 쥐와 선인장이 나타날지 예측하기 더 어려워졌습니다.

### 확인하는 법
- 현재 실행 중인 웹 페이지 `http://localhost:3000` 을 새로고침 하시면 새롭게 바뀐 다이나믹 시스템을 바로 플레이 하실 수 있습니다!
