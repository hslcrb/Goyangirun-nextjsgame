# ✨ 신비로운 발견: '믿음'의 오토파일럿 이스터 에그 구현 완료

고양이의 발걸음을 지켜주는 보이지 않는 손, **'믿음'** 키워드를 통한 오토파일럿 기능이 추가되었습니다.

## 🗝️ 활성화 방법 (Easter Egg)

게임 도중 혹은 대기 중에 키보드로 다음 키워드 중 하나를 입력해 보세요:
- **English**: `believe`, `Believe`
- **Korean**: `믿어`, `믿는다`, `난 널 믿어`, `난널믿어`, `널 믿는다`

## 🌟 주요 기능 및 연출

1. **자동 점프 (Autopilot AI)**:
   - 키워드가 활성화되면 고양이가 다가오는 선인장을 스스로 감지하고 최적의 타이밍에 점프합니다.
   - 플레이어는 이제 고양이가 안전하게 달리는 모습을 지켜보며 음악을 감상할 수 있습니다.

2. **감성적인 문구**:
   - 활성화 시 화면 중앙에 **"누군가 나를 믿어준다는 것."** 이라는 문구가 부드럽게 나타났다가 사라집니다. 이는 단순한 기능 활성화 알림이 아닌, 게임의 감성적인 서사를 완성하는 장치입니다.

3. **해제 조건 (Reset)**:
   - 사용자의 의도대로, **게임 오버**가 발생하거나 **새로고침**을 하면 오토파일럿 모드는 초기화됩니다. (다시 한 번 믿음을 주어야 활성화됩니다.)

### 확인 방법
- 게임을 시작한 후 키를 하나씩 눌러 `believe`를 입력해 보세요.
- 화면 상단에 묘한 여운을 주는 문구가 나타나며 고양이가 선인장을 알아서 피하기 시작합니다. 🐈💖
